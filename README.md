# 🤖 GRIDBOT — Grid Long Strategy Optimizer

Инструмент для оптимизации и бэктестинга лонговой сеточной торговой стратегии на криптовалютных рынках. Поддерживает случайный поиск (Random Search) с элитным направлением (Elite Guided Search) для нахождения оптимальных параметров сетки.

## ⚙️ Логика стратегии

1. Сетка из **N лимитных лонг-ордеров** выставляется ниже текущей цены.
2. Каждый ордер имеет **индивидуальный шаг** (% от точки входа) и **индивидуальный размер** (USDT).
3. При касании уровня — ордер исполняется и добавляется к позиции.
4. **TP** рассчитывается от средневзвешенной цены входа всей позиции.
5. При достижении TP — позиция закрывается, сетка **перезапускается** от новой цены.
6. Данные: **1-минутный таймфрейм** для максимальной точности.

## 🎛️ Параметры оптимизации

| Параметр | Диапазон | Описание |
|---|---|---|
| `n_orders` | 1..40 | Количество ордеров в сетке |
| `steps[i]` | 0.1%..8% каждый | Индивидуальный шаг i-го ордера |
| `sizes[i]` | 5$..500$ каждый | Индивидуальный размер i-го ордера |
| `tp_pct` | 0.3%..8% | % тейк-профита от avg_entry |

**Режимы шагов:** `linear`, `geometric`, `random`, `front_heavy`, `back_heavy`

**Режимы размеров:** `flat`, `linear_increase`, `geometric_increase`, `pyramid`, `random`

---

## 📊 Критерий оценки (Score)

```
Score = (PnL / MaxDD) * speed_bonus * profit_factor_bonus
speed_bonus = 1 / log(avg_minutes + 2)
```

Формула одновременно максимизирует PnL, минимизирует максимальную просадку (MaxDD) и среднее время удержания позиции.

Каждые **400 итераций** пространство поиска автоматически сужается вокруг лучших найденных параметров.

## 🖥️ Запуск

### Базовый запуск

```bash
python main_grid.py --file /path/to/BYBIT_BTCUSDT_LINEAR_2020_2026.csv
```

### Полный пример

```bash
python main_grid.py \
    --file /path/to/BYBIT_BTCUSDT_LINEAR_2020_2026.csv \
    --symbol BTCUSDT \
    --iters 3000 \
    --capital 1000 \
    --commission 0.0006 \
    --out results/BTCUSDT \
    --seed 42
```

### Другая монета

```bash
python main_grid.py \
    --file /path/to/BYBIT_ETHUSDT_LINEAR_2021_2026.csv \
    --symbol ETHUSDT \
    --iters 2000 \
    --capital 1000
```

### Ресемплинг (ускорение через более крупный таймфрейм)

```bash
python main_grid.py --file data.csv --tf 5
```

### Запуск бэктеста по сохранённым параметрам

```bash
python main_grid.py \
    --file data.csv \
    --backtest results/BTCUSDT/grid_opt_BTCUSDT_best_params.json
```

## 📂 Результаты

После завершения оптимизации в папке `--out` появятся:

| Файл | Описание |
|---|---|
| `grid_opt_{SYMBOL}.csv` | Все результаты оптимизации |
| `grid_opt_{SYMBOL}_best_params.json` | Топ-20 конфигураций с полными параметрами |

| `grid_heatmap_{SYMBOL}.png` | Тепловая карта n_orders vs tp_pct |
